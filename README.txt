veZo - Gerenciador de Hábitos e Atividades

Descrição:

veZo é um organizador de hábitos e atividades desenvolvido em Python para ser utilizado via terminal. O objetivo do projeto é auxiliar usuários na criação e manutenção de hábitos saudáveis, organização de tarefas e acompanhamento do progresso pessoal.

Funcionalidades:

* Cadastro de usuários com senha protegida
* Login seguro
* Adição de atividades personalizadas:
   * Hábitos recorrentes (diários ou em dias específicos)
   * Tarefas únicas
   * Checklists
* Edição e remoção de atividades
* Marcar atividades como concluídas
* Acompanhamento do histórico de atividades

Requisitos:

Para executar o veZo, certifique-se de ter o Python instalado em seu sistema.
Também é necessário instalar a biblioteca colorama para melhor experiência visual no terminal. Instale com o seguinte comando:    pip install colorama

Uso:

Ao iniciar o programa, o menu principal será exibido com as opções:
1. Cadastro de Usuário
2. Login
3. Adicionar Atividade
4. Mostrar Atividades
5. Editar ou Excluir Atividades
6. Marcar Atividade como Concluída
7. Visualizar Histórico
8. Sair

Basta seguir as instruções no terminal para utilizar as funcionalidades, bom uso!
